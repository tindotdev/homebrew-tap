# tindotdev/tap

Homebrew formulae for [tindotdev](https://github.com/tindotdev) projects. Pre-built binaries for macOS and Linux.

## Install

```bash
brew tap tindotdev/tap
brew install tindotdev/tap/dictate-cli
```
